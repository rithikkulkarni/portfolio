Overview

My two-month internship at Syntelli Solutions marked a turning point in my journey toward a career in software engineering. As my first professional exposure to the field, I entered with a hunger to see behind-the-scenes development of software solutions. Syntelli provided me with the ideal environment to explore this.

The first few days were an introduction to the professional standards and design principles that guide the software engineering industry. I was introduced to front-end wireframing, user task flows, and the importance of careful planning before any coding begins. Each lesson was reinforced with real-world examples, highlighting how thoughtful design can enhance the user experience. These insights provided a stable foundation for my project and noted the importance of planning and organization.

Once I had a firm grasp of these concepts, I was assigned my primary project—a section of a customer solution that Syntelli was developing. My task was to map out and create a full Java application out of this section of a customer’s desired product, which I would ultimately present to the team working on this solution, as well as Syntelli's VP and CTO. Tackling this project required not only technical skills but also a deep understanding of the user experience. I designed, developed, and tested the application, adhering to the standards I’d been introduced to in those initial days. The challenge of crafting a polished product for presentation was both thrilling and demanding.

Working on this project provided me with industry skills and insights. I learned to see software engineering as a structured process where each design decision impacts the final user experience. Wireframing and task flow mapping taught me to consider the user journey from start to finish. Additionally, I strengthened my technical skills in Java, gaining confidence in my ability to transform designs into a functional application.

Reflecting on my time at Syntelli, I realize how much this internship solidified my passion for software engineering. The exposure to industry standards gave me an appreciation for the creativity involved in building impactful software. Learning how to design with the user in mind and experiencing the entire development cycle from concept to presentation reinforced my dedication to pursuing this career path.


Key Skills

Software Design: Developed a full Java application for a customer solution, applying industry-standard practices in design, coding, and testing to ensure a robust and user-friendly final product.
UI/UX Design: Utilized front-end wireframing and task flow mapping to optimize the user experience, creating an intuitive design layout based on user-centered principles learned through industry examples.
Java: Used Java to build, implement, and debug application features, gaining hands-on experience with the language while ensuring code quality and functionality aligned with project requirements.
